- 文件不存在
- 用户输入了非数字
- 网络请求失败

如果处理不了，程序直接报错退出。

用 `try...except` 可以优雅应对。



### 安全地读取文件

---

```python
def read_file_safely(filename):
    try:
        with open(filename, 'r', encoding) as f:
            return f.read
        except FileNotFoundError:
            print(f"❌ 错误：文件 '{filename}' 找不到！")
        return None
    except Exception as e:
        print(f"❌ 发生未知错误: {e}")
        return None

# 测试
content = read_file_safely("不存在的文件.txt")
if content is not None:
    print(content)
```



### 常见异常类型

---

- `FileNotFoundError`：文件没找到
- `ValueError`：类型转换失败（如 `int("abc")` ）
- `KeyError`：字典里没有这个键



### 代码

---

编写一个函数 safe_divide(a, b)，安全地做除法。如果除数是0，提示“不能除以零”，并返回None。

```python
def safe_divide(a, b):
    try:
        return a/b
    except ZeroDivisionError:
        print("❌ 不能除以零！")
        return None

# 测试
print(safe_divide(10, 2))  # 5.0
print(safe_divide(10, 0))  # ❌ 不能除以零！ → None
```
