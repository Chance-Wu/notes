

函数把一段代码“打包”，重复使用。

```python
def greet(name):
    return f"你好，{name}！今天要写什么工具？"

print(greet("chance"))  # 输出：你好，chance！今天要写什么工具？
```



### 文件读写

---

假设有一个 `notes.txt` 文件，想读取内容：

```python
# 读取文件
with open("notes.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)
```

写入文件（比如生成报告）：

```python
# 写入文件
with open("report.txt", "w", encoding="utf-8") as f:
    f.write("今日数据分析完成！\n共处理 100 行数据。")
```

>注意：`with open...` 会自动关闭文件，避免出错。



创建一个函数，把消息追加写入 `my_tool.log` 文件（提示：用 `"a"` 模式）。

```python
def save_log(message):
    with open("my_tool.log", "a", encoding="utf-8") as f:
        f.write(message + "\n")

# 测试
save_log("工具启动成功")
save_log("处理完成")
```
