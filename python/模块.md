随着工具变复杂，所有代码写在一个文件会很乱。

**模块 = 一个 `.py` 文件**，你可以把它“导入”到其他文件使用。



### 创建一个工具包

---

```text
my_tools/
|—— main.py
|____ file_utils.py
```

file_utils.py

```python
def count_lines(filename):
    """统计文件行数"""
    try:
        with open(filename, 'r' , encoding='utf-8') as f:
            return len(f.readlines())
        except Exception as e:
            print(f"读取文件出错: {e}")
        return 0

def backup_file(src, dst):
    """简单备份文件"""
    try:
        with open(src, 'r', encoding='utf-8') as f_in:
            with open(dst, 'w', encoding='utf-8') as f_out:
                f_out.write(f_in.read())
        print(f"✅ 已备份 {src} → {dst}")
    except Exception as e:
        print(f"备份失败: {e}")
```

main.py

```python
# 导入我们写的模块
import file_utils

# 使用里面的函数
lines = file_utils.count_lines("notes.txt")
print(f"文件共有 {lines} 行")

file_utils.backup_file("notes.txt", "notes_backup.txt")
```
