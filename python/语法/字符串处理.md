Python 字符串和 Java 一样，也是**不可变对象**（Immutable）。但 Python 提供了一些非常独特的工具，可以像切面包一样处理字符串。

#### 切片

切片的标准格式是：`object[start:stop:step]`

- **start**: 起始索引（包含），默认为 0。
- **stop**: 结束索引（不包含），默认为序列长度。
- **step**: 步长，默认为 1。

#### 负数索引（java 无）

支持从右往左数，最后一个元素的索引是-1，倒数第二个是-2，依次类推。

假设我们有一个字符串 `s = "JavaBackend"`：

- **获取前 4 个字符**:
  - Python: `s[:4]` （等同于 `s[0:4:1]`）
  - 结果: `"Java"`
- **获取最后 7 个字符**:
  - Python: `s[-7:]`
  - 结果: `"Backend"`
- **每隔一个字符取一个（跳跃截取）**:
  - Python: `s[::2]`
  - 结果: `"JvBced"`

#### f-string 格式化

Python 3.6+ 引入的最受欢迎的特性。相比 Java 的 `String.format("%s", var)` 或者繁琐的 `+` 号拼接，f-string 让代码极具可读性：

```python
name = "Javaer"
print(f"Hello, {name.upper()}!") # 直接在 {} 里写表达式
```

#### strip

移除字符串头尾指定的字符（默认为空格或换行符）或字符序列。

```python
str.strip([chars]);
```

- lstrip
- rstrip

#### in

**in** 操作符用于判断键是否存在于字典中，如果键在字典 dict 里返回 true，否则返回 false。

**not in** 操作符刚好相反，如果键在字典 dict 里返回 false，否则返回 true。

#### join

用于将序列中的元素以指定的字符连接生成一个新的字符串。

```python
seq = ('a', 'b', 'c', 'd')
'-'.join(seq)
```

#### replace

把字符串中的 old（旧字符串） 替换成 new(新字符串)，如果指定第三个参数max，则替换不超过 max 次。

```python
str.replace(old, new[, max])
```
