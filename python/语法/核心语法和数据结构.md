### 一、变量 & 数据类型

```python
a = 10          # int
b = 3.14        # float
c = "hello"     # str
d = True        # bool
```

Python 是动态类型。

>使用 `type()` 查看变量类型。
>
>```python
>print(type(a))	# <class 'int'>
>print(type(b))	# <class 'float'>
>print(type(c))	# <class 'str'>
>print(type(d))	# <class 'bool'>
>```

### 三、字典 dict

---

```python
user = {
    "id": 1,
    "name": "chance"
}
```

#### 3.1 取值

```python
user["name"]

user.get("name")
```

#### 3.2 遍历

```python
for k, v in user.items():
    print(k, v)
```
