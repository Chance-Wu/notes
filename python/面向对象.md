### 创建类

---

使用 class 语句来创建一个新类，class 之后为类的名称并以冒号结尾：

```python
class ClassName:
    """
    类的帮助信息
    """
    class_suite # 类体
```

类的帮助信息可以通过 `ClassName.__doc__` 查看。

`class_suite` 由类成员，方法，数据属性组成。











































### 一、构造函数与 self（java 的 this）

---

在 java 中，构造函数与类名相同。在 python 中，构造函数永远叫 `__init__` 。

`self` 必须**显式写在方法的第一个参数位**。它等同于 java 的 `this`，代表**实例本身**。

| 特性     | java                             | python                     |
| -------- | -------------------------------- | -------------------------- |
| 定义类   | public class User { ... }        | `class User:`              |
| 构造函数 | public User(String name) { ... } | `def __init__(self, name)` |
| 实例化   | User user = new User("Alice")    | `user = User("Alice")`     |



### 二、 没有真正的封装

---

python 中没有硬性的访问控制。遵循“君子协定”：

- `_name`（**单下划线**）：相当于 `protected`，告诉别人“这是一个私有属性，请尽量不要在类外部直接访问”。
- `__name`（**双下划线**）：触发“名称修饰”，让外部更难访问，模拟 `private`。

```python
class Person:
    def __init__(self, name):
        self.name = name  # 直接赋值，默认就是 public

    def say_hello(self):
        print(f"Hello, {self.name}")
```

>Python没有真正的私有属性，下划线只是给开发者提示。



### 三、继承

---

python **支持多重继承**。

```python
class Admin(User, Permissions): # 同时继承两个类
    def __init__(self, name, level):
        super().__init__(name)  # 同样使用 super() 调用父类
        self.level = level
```

```python
class Counter:
    count = 0  # 它是直接写在类里，而不是 __init__ 里的
```

>在类体中定义的变量就是**类变量**，类似于java的 `static` 字段。



### 四、@property 装饰器

---

python中，直接访问属性（user.age）。如果需要在这个属性访问时加入逻辑（比如校验年龄不能小于0），使用`@property`。

```python
class User:
    def __init__(self, age):
        self._age = age # 下划线暗示这是私有的

    @property
    def age(self):
        return self._age

    @age.setter
    def age(self, value):
        if value < 0:
            raise ValueError("年龄不能为负数！")
        self._age = value

# 使用时，看起来像是在操作普通变量，但背后触发了逻辑
user = User(25)
user.age = 30  # 自动调用 setter
```



### 五、魔术方法

---

以**双下划线开头和结尾**的方法被称为“Dunder Methods”（Double Under）。

```python
class User:
    def __init__(self, name):
        self.name = name

    def __str__(self):
        return f"User(name={self.name})"

user = User("Pythoner")
print(user)  # 输出: User(name=Pythoner)
```



### 六、dataclass

---

```python
from dataclasses import dataclass

@dataclass
class UserDTO:
    id: int
    name: str
    email: str

# 自动帮你写好了 __init__, __repr__, __eq__ 等所有方法！
user = UserDTO(1, "Alice", "alice@example.com")
print(user)
```
