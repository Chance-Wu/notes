

### 一、构造函数与 self（java 的 this）

---

在 java 中，构造函数与类名相同。在 python 中，构造函数永远叫 `__init__` 。

`self` 必须**显式写在方法的第一个参数位**。它等同于 java 的 `this`，代表**实例本身**。

| 特性     | java                             | python                     |
| -------- | -------------------------------- | -------------------------- |
| 定义类   | public class User { ... }        | `class User:`              |
| 构造函数 | public User(String name) { ... } | `def __init__(self, name)` |
| 实例化   | User user = new User("Alice")    | `user = User("Alice")`     |



### 二、 没有真正的封装

---

python 中没有硬性的访问控制。遵循“君子协定”：

- `_name`（**单下划线**）：相当于 `protected`，告诉别人“这是一个私有属性，请尽量不要在类外部直接访问”。
- `__name`（**双下划线**）：触发“名称修饰”，让外部更难访问，模拟 `private`。
