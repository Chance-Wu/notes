### 一、安装prometheus

---

1. 通过homebrew安装

   ```shell
   brew services start prometheus
   ```

2. 启动prometheus

3. 查看链接：http://localhost:9090/

4. 重启prometheus

   ```shell
   brew services restart prometheus
   ```



### 二、安装Grafana

---

1. 通过homebrew安装

   ```shell
   brew install grafana
   ```

2. 启动grafana

   ```shell
   brew services start grafana
   ```

3. 访问http://localhost:3000/
