>把配置文件中的值注入到工具类。

```java
public class PayPlatformPageUtil {
    private static String tenantCode;

    @Value("${tenantCode}")
    private String tenant;

    @PostConstruct
    public void init() {
        PayPlatformPageUtil.tenantCode = tenant;
    }

    public static String getEnumKeyByTenantCode(String enumName, String value){
        if(null != tenantCode){
            System.out.println(tenantCode);
        }
    }

}
```
