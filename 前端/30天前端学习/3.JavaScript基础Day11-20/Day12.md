### 一、函数基础

---

#### 1.1 函数声明

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
console.log(greet("Chance")); // "Hello, Chance!"
```

特点：

- 存在函数提升（可在定义前调用）
- 拥有函数名标识符
- 适合定义主要功能函数
