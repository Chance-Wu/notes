>学习目标：
>
>- 掌握语义化表格结构（table/tr/th/td）的正确写法
>- 理解表单核心元素（form/input/button/select/textarea）的用途与属性
>- 能独立创建带基本验证功能的用户注册表单
>- 区分展示型数据（表格）与交互型数据（表单）的使用场景



### 一、HTML 表格语义结构

---

#### 1.1 表格必须标签体系

| 标签      | 作用       | 必须嵌套关系    | 示例                                    |
| --------- | ---------- | --------------- | --------------------------------------- |
| `<table>` | 表格容器   | 最外层          | `<table>`                               |
| `<thead>` | 表头区域   | table 内        | `<thead><tr><th>姓名</th></tr></thead>` |
| `<tbody>` | 表体区域   | tabel 内        | `<tbody><tr><td>张三</td></tr></tbody>` |
| `<tr>`    | 表行       | thead/tdbody 内 | `<tr>`                                  |
| `<th>`    | 表头单元格 | tr 内           | `<th>年龄</th>`                         |
| `<td>`    | 数据单元格 | tr 内           | `<td>25</td>`                           |

#### 1.2 关键规范

- 必须包含 `<thead>` 和 `<tbody>`（即使只有一行数据）
- 表头用 `<th>` ，数据用 `<td>`（屏幕阅读器会区分读法）
- 错误：直接在 `<table>` 内写 `<tr>`（缺少 thead/tbody）
- 错误：用 `<div>` 模拟表格（破坏语义和可访问性）



### 二、HTML 表单核心元素

---

#### 2.1 表单基础结构

```html      
<!-- 表单容器 -->
<form action="/submit" method="POST">
    <!-- 表单控件 -->
    <button type="submit">提交</button>
</form>
```

- action：数据提交地址
- method：提交方式（GET/POST）

#### 2.2 核心输入类型与属性

| 元素         | 类型     | 必需属性 | 语义说明 | 示例                                                 |
| ------------ | -------- | -------- | -------- | ---------------------------------------------------- |
| `<input>`    | text     | name     | 普通文本 | `<input type="text" name="username">`                |
| `<input>`    | email    | name     | 邮箱格式 | `<input type="email" name="email">`                  |
| `<input>`    | password | name     | 密码隐藏 | `<input type="password" name="pwd">`                 |
| `<input>`    | tel      | name     | 电话号码 | `<input type="tel" name="phone">`                    |
| `<select>`   | -        | name     | 下拉选择 | `<select name="city"><option>北京</option></select>` |
| `<textarea>` | -        | name     | 多行文本 | `<textarea name="bio"></textarea>`                   |
| `<button>`   | submit   | type     | 提交按钮 | `<button type="submit">注册</button>`                |

####  2.3 关键属性

- `name` 属性：必须设置！用于后端识别字段（如 name="email"）
- `placeholder`：提示文字（非替代 label）
- `required`：必填验证（浏览器自动校验）

#### 2.4 语义化最佳实践

每个输入框必须有 `<label>`：

```html
<label for="userName">姓名：</label>
<input type="text" id="userName" name="userName">
```



### 三、表单 vs 表格的本质区别

| 特性     | 表格（table）        | 表单（form）         |
| -------- | -------------------- | -------------------- |
| 目的     | **展示数据（静态）** | **收集数据（交互）** |
| 用户操作 | 查看/排序            | 输入/选择/提交       |
| 语义重点 | 数据关系             | 字段含义             |
| 示例     | 成绩单、价格表       | 注册页、搜索框       |



### 四、实践任务

---

#### 4.1 步骤1：创建课程成绩表格

在 index.html 的 `<main>` 中添加：

```html
<h2>课程成绩</h2>
<!-- 表格容器 -->
<table>
    <!-- 表头区域 -->
    <thead>
        <!-- 表行 -->
        <tr>
            <!-- 表头单元格 -->
            <th>课程</th>
            <th>学分</th>
            <th>成绩</th>
        </tr>
    </thead>
    <!-- 表体区域 -->
    <tbody>
        <tr>
            <!-- 数据单元格 -->
            <td>HTML 与 CSS 基础</td>
            <td>3</td>
            <td>92</td>
        </tr>
        <tr>
            <td>JavaScript 入门</td>
            <td>4</td>
            <td>88</td>
        </tr>
    </tbody>
</table>
```

#### 4.2 步骤2：创建用户注册表单

在表格下方添加：

```html 
<h2>用户注册</h2>
<!-- 表单容器 -->
<form>
    <div <!-- 表单控件 -->
        <label for="reg-name">姓名：</label>
        <input type="text" id="reg-name" name="fullName" required>
    </div>

    <div>
        <label for="reg-email">邮箱：</label>
        <input type="email" id="reg-email" name="email" required placeholder="example@admin.com">
    </div>

    <div>
        <label for="reg-pws">密码：</label>
        <input type="password" id="reg-pws" name="password" required minlength="6">
    </div>

    <div>
        <label for="reg-city">城市：</label>
        <select id="reg-city" name="city" required>
            <option value="">请选择</option>
            <option value="beijing">北京</option>
            <option value="shanghai">上海</option>
            <option value="guangzhou">广州</option>
        </select>
    </div>

    <div>
        <label for="reg-bio">个人简历：</label>
        <textarea id="reg-bio" name="bio" rows="3" placeholder="简单介绍自己..."></textarea>
    </div>

    <button type="submit">立即注册</button>
</form>
```

#### 4.3 步骤3：验证关键点

- 表格验证：
  - 检查是否包含`<thead>`/`<tbody>`
  - 确认表头用`<th>`（浏览器默认加粗居中）
- 表单验证：
  - **点击label是否能聚焦输入框**
  - 不填必填项直接提交 → 浏览器应弹出验证提示
  - 邮箱格式错误时提交 → 应提示"请输入有效邮箱"



### 五、关键建议

---

#### 5.1 表格不要用于布局

- 错误：用table做页面整体布局（2000年代做法）
- 正确：仅用于展示行列数据（如成绩单、价格表）

#### 5.2 表单验证优先用 HTML 原生属性

- `required`（必填）
- `minlength`/`maxlength`（长度限制）
- `type="email"`（格式验证）

####  5.3 label 的两种写法

```html
<!-- 显式关联（推荐） -->
<label for="id">文本</label>
<input id="id">

<!-- 隐式包裹 -->
<label>
  文本
  <input>
</label>
```
