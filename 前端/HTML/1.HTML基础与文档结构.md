### 一、HTML 基础概念

---

#### 1.1 什么是 HTML

`超文本标记语言（Hyper Markup Language）`，用于构建网页结构的标记语言（不是编程语言）。

#### 1.2 关键理解

- HTML 由标签 Tags 和内容组成，标签用 `<>` 包裹
- 浏览器通过解析 HTML 标签渲染页面内容
- 示例：<p>这是一个段落</p> 中<p>是标签，这是一个段落是内容



### 二、HTML 文档结构

---

#### 2.1 必需标签解析

| 标签              | 作用                                       | 位置         | 示例              |
| ----------------- | ------------------------------------------ | ------------ | ----------------- |
| `<!DOCTYPE html>` | 声明文档类型，告诉浏览器这是HTML5文档      | 文档最顶部   | `<!DOCTYPE html>` |
| `<html>`          | HTML文档的根容器                           | 包裹所有内容 | `<html>`          |
| `<head>`          | 页面头部区域，包含元数据（不显示在页面上） | `<html>`内部 | `<head>`          |
| `<body>`          | 页面主体内容区域（实际显示内容）           | `<html>`内部 | `<body>`          |

#### 2.2 关键细节

- `<!DOCTYPE html>`必须在文档最开头，否则浏览器可能进入怪异模式
- `<head>`中必须包含`<title>`（页面标题）和`<meta charset="UTF-8">`（字符编码）
- `<body>`是唯一显示内容的容器



### 三、HTML 语法规范

---

#### 3.1 标签规则

- 开始标签：`<tag>`
- 结束标签：`</tag>`
- 空标签：`<br/>`（自闭合标签）



### 四、实践

---

#### 4.1 创建 HTML 文件

```html
<!DOCTYPE html>
<html>

<head>
    <title>HTML Structure</title>
    <meta charset="utf-8" />
</head>

<body>
    <!-- 页面内容 -->
</body>

</html>
```

#### 4.2 验证结构

1. 保存文件后，用浏览器打开（双击`index.html`）
2. 按`F12`打开开发者工具 → Elements面板
3. 检查浏览器渲染的HTML结构是否符合：
   - 文档类型声明正确
   - 标签嵌套关系正确（head/body在html内部）
   - 无缺失标签

#### 4.3 添加测试内容

- 在 `<body>` 中添加以下内容：

  ```html
  <h1>HTML 结构</h1>
  <p>第一个网页</p>
  <img src="../img/image.jpg" alt="示例图片">
  ```

- 重新加载页面，观察：

  - `<h1>`显示为大标题
  - `<p>`显示为段落
  - `<img>`加载占位图
