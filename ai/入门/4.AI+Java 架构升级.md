## RAG（检索增强生成）

---

解决 AI 不懂你业务的问题

结构：

```
用户问题 → 检索数据库 → 拼接上下文 → AI回答
```

java 实现：

- 存业务数据（MySQL / ES）
- 检索
- 拼 prompt



## Agent（自动执行）

---

例子：帮我查订单并发邮件

Agent 流程：

1. 查数据库
2. 调接口
3. 发邮件
